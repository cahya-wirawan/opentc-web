Open Text Classification (OpenTC)

{# Load the tag library #} {% load bootstrap3 %}

This is a demo website to show the capability of OpenTC. The OpenTC server is running in the background listening to the port 3333. The website is running Django, and forward the prediction request from the web client to the OpenTC server, and forward back again the result from the server to the web client as json response.

Currently it supports Bayes, Support Vector Machine (SVM), and Convolutional Neural Network (CNN) as classifier and they were trained with 4 newsgroups (alt.atheis, com.graphics, sci.med, soc.religion.christian) from the 20newsgroup dataset.
